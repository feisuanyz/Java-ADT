```

key：tab面板上的key

title：选项卡头显示文字
```

```
示例：
 {
     "msg": "调用成功！",
     "result": "1",
     "code": "000000",
     "data": [
         {
            "key": "Tabs1",
             "title": "动态标签页1",
         },
         {
             "key": "Tabs2",
             "title": "动态标签页2",
         }
     ]
}
```
